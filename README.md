# Tuition Reimbursement System

## Project Description

A Tuition reimbursement system that allows a user to send in a request and elevated users to approve or deny the request for reimbursement.

## Technologies Used

* Python3 - version 3.9
* AWS Database
* JavaScript
* Cucumber
* Selenium
